# STUDENT-DATABASE
The Student Database Management System is a Python program designed to handle basic student record operations in a simple and interactive way. It allows users to add new students with their USN, name, and marks in three subjects (Python, Java, and C), and then automatically calculates their percentage, grade, and individual subject pass/fail status. The system is built around a menu‑driven interface, making it easy to navigate through options such as adding, displaying, searching, and deleting student records. Internally, the program uses dictionaries to store student data, ensuring quick access and efficient management of information. By combining functions, loops, and conditionals, this project demonstrates how Python can be used to implement a small‑scale database system and highlights key programming concepts like modularity, CRUD operations, and result evaluation. It serves as a practical example for beginners to understand how to structure code for real‑world applications while keeping it simple and user‑friendly.
